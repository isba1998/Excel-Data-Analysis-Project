# Excel-Data-Analysis-Project

## Overview

This project uses Microsoft Excel to analyze a biological dataset containing approximately 1,000 abalone records. The dataset includes physical measurements, weight measurements, gender, number of rings, collection information, water region, and temperature data.

## Analysis Performed

The workbook includes:

* **Data Categorization** – Grouped records based on defined criteria.
* **Statistical Analysis** – Calculated descriptive statistics, including mean, variance, minimum, and maximum for key biological measurements.
* **Economic Value Analysis** – Analyzed estimated economic value and compared value categories.
* **Search Functions** – Used Excel search and lookup functions to retrieve individual records using an Abalone ID.
* **Regional Analysis** – Summarized and compared biological measurements across geographic water regions.
* **Optimal Allocation** – Applied optimization concepts to evaluate allocation scenarios across geographic regions.
